# 🍕 Pizza Sales Analysis Dashboard

## 📌 Project Overview

This project is an interactive Pizza Sales Analysis Dashboard developed using **Oracle SQL** and **Microsoft Power BI**. The dashboard provides valuable business insights by analyzing one year of pizza sales data. It helps understand sales performance, customer ordering patterns, product popularity, and overall business trends through interactive visualizations.

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Oracle SQL
- Power Query
- DAX (Data Analysis Expressions)

---

## 📊 Dashboard Features

### 🏠 Home Dashboard

- Total Revenue
- Total Orders
- Total Pizzas Sold
- Average Order Value
- Average Pizzas per Order
- Daily Order Trend
- Monthly Order Trend
- Sales by Pizza Category
- Sales by Pizza Size
- Busiest Days & Months
- Interactive Filters (Slicers)

### ⭐ Best/Worst Seller Dashboard

- Top 5 Pizzas by Revenue
- Top 5 Pizzas by Quantity Sold
- Top 5 Pizzas by Total Orders
- Bottom 5 Pizzas by Revenue
- Bottom 5 Pizzas by Quantity Sold
- Bottom 5 Pizzas by Total Orders

---

## 📈 Key Business Insights

- 💰 Total Revenue: **817.86K**
- 🍕 Total Pizzas Sold: **49,574**
- 🛒 Total Orders: **21,350**
- 💵 Average Order Value: **38.31**
- 🍕 Average Pizzas per Order: **2.32**
- 📅 Highest Orders: **Friday & Saturday Evenings**
- 📆 Highest Monthly Orders: **July**
- 🥇 Classic Pizza Category generated the highest sales.
- 📏 Large Size pizzas were the most preferred by customers.
